# Perspective Corrector

Corrects perspective of images with intuitive UI.

## Overview

Perspective Corrector is a web-based application that helps you fix skewed or angled photos. Whether you've captured a document at an angle, taken a photo of a whiteboard, or snapped a picture of an object from the side, this tool uses advanced image processing to straighten and correct the perspective to make your images look perfectly aligned.

## Features

- 🖼️ **Easy Image Upload** - Simply drag and drop or select an image to get started
- 🎯 **Intuitive Controls** - Adjust perspective with an easy-to-use interface
- 🔄 **Real-time Preview** - See your changes as you make them
- 📥 **Multiple Format Support** - Works with common image formats (JPEG, PNG, etc.)
- 💾 **Download Results** - Save your corrected images with a single click
- ⚡ **Fast Processing** - Lightweight and responsive performance
- 🎨 **Clean UI** - Modern, user-friendly interface

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs entirely in your browser

### Usage

1. Open the application in your web browser
2. Upload an image by clicking the upload button or dragging and dropping
3. Use the control points or adjustment tools to correct the perspective
4. Preview your corrected image in real-time
5. Download your corrected image when satisfied

## How It Works

The application uses canvas-based image manipulation to:
1. Load your image onto an editable canvas
2. Allow you to define the perspective correction by adjusting corner points or angles
3. Apply geometric transformation to straighten the image
4. Render the corrected result for download

## Tips for Best Results

- Use images with clear edges or lines for easier perspective correction
- Ensure good lighting in your original image
- For documents, try to capture the entire page including edges
- Experiment with subtle adjustments for the best results
