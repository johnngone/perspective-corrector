# Perspective Corrector

Corrects perspective of images with intuitive UI and pixel-perfect accuracy.

## Overview

Perspective Corrector is a web-based application that helps you fix skewed photos at the pixel level.

## Features

- **Local Processing** - All processing happens locally in your browser
- **Easy Image Upload** - Simply drag and drop or select an image to get started
- **Auto Guide** - Automatically place guides in image (experimental)
- **Intuitive Controls** - Adjust perspective with an easy-to-use interface down to the pixel level.
- **Crop Image** - Crop images to guides
- **Aspect Ratio Scaling** - Specify aspect ratio of image to avoid stretched look
- **Export Images** - Save your corrected images with a single click
- **Save Project** - Save your coordinates to JSON with a single click

## Usage

- Serve files on webhost or run locally on your PC
